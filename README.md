## 🚀 **Quick Start**

### **Prerequisites**
- Node.js 18+ 
- npm 8+
- Git

### **Installation**
```bash
# Clone the repository
git clone https://github.com/FacundoLuna16/Portafolio.git
cd Portafolio

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:3000
```

### **Development Commands**
```bash
# Development
npm run dev              # Start dev server
npm run build           # Production build
npm run start           # Start production server

# Quality Assurance
npm run lint            # ESLint check
npm run lint:fix        # Fix ESLint issues
npm run type-check      # TypeScript check
npm run health          # Full health check

# Performance
npm run analyze         # Bundle analysis
npm run size            # Bundle size check
npm run audit:performance  # Lighthouse audit

# PWA
npm run icons:generate  # Generate PWA icons
npm run optimize:images # Optimize images

# Deployment
npm run deploy          # Automated deployment
npm run deploy:preview  # Preview deployment
npm run deploy:prod     # Production deployment
```

## 🚀 **Deployment**

### **Automated Deployment**
```bash
# Run complete deployment pipeline
npm run deploy
```

The deployment script includes:
1. **🔍 Pre-deployment checks** (Node.js, dependencies, files)
2. **🧹 Code quality** (TypeScript, ESLint, security audit)
3. **⚡ Build optimization** (PWA icons, image optimization)
4. **📊 Performance verification** (Lighthouse audit)
5. **🌍 Vercel deployment** with automatic URL
6. **✅ Post-deployment testing** (accessibility, PWA features)

### **Manual Deployment**
```bash
# Preview deployment
npm run deploy:preview

# Production deployment  
npm run deploy:prod
```
